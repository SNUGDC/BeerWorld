---------------------------------------
Find Asset References ver 1.0
---------------------------------------
* This editor extension is allow find the referenced GameObjects of the selected object in currently scene and project.

---------------------------------------
* USAGE

1. Open context menu by right click on any object's inspector.

2. Select 'FN-Find References'.

3. If you launch first time or AssetCache is empty, you will see a UpdateCache dialog.(you can ignore setting in "Asset References Window").

4. "Asset Cache update NOW!" button in "Asset References Window" is make asset cache for fast finding references.
